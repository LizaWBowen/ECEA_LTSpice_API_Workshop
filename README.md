# LTspice + Python Workshop  
## Monte Carlo Circuit Reliability Analysis

This workshop demonstrates how to use **Python to automate LTspice simulations** and perform a **Monte Carlo tolerance analysis** on an analog circuit.

Students will learn how to:
- Control LTspice from Python
- Run hundreds of simulations automatically
- Model real-world component tolerances
- Analyze statistical variation in circuit performance
- Generate automated PDF reports

---

# 🚀 Project Overview

In real-world circuits, components are not ideal:

- Resistors: ±5% tolerance  
- Capacitors: ±10% tolerance  

These variations affect circuit behavior.

In this workshop, we will:
1. Build a simple RC low-pass filter in LTspice  
2. Randomize component values using Python  
3. Run **200+ simulations automatically**  
4. Measure cutoff frequency for each case  
5. Generate a statistical report
